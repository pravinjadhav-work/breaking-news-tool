# 📰 Breaking News Poster Generator

A professional web-based tool to create viral **Instagram-style Breaking News posts**.

Inspired by pages like:
- IndianLast24Hr
- WorldInLast24Hr

These types of pages have **millions of followers** and use visually strong news templates — this tool recreates that workflow.

---

## 🚀 Features!

- 🖼 Upload any image
- 🔠 Bold BREAKING headline system
- 🎯 Highlight specific words with red background
- 📅 Custom date badge (day + month)
- 🏷 Watermark branding
- 🎚 Fine control over:
  - Font size
  - Spacing
  - Fade shadow
  - Image positioning
- 📥 Download high-quality PNG instantly

---

## 🎯 Use Cases

- Instagram news pages
- Content creators
- Meme/news pages
- Portfolio projects
- Social media automation

---

## 🛠 Tech Stack

- HTML5
- CSS3 (Glassmorphism UI)
- Vanilla JavaScript (Canvas API)
